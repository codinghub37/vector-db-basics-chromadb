# 🧠 Vector Database Basics with ChromaDB

A practical introduction to **Vector Databases** using **ChromaDB**. This project demonstrates how vector databases store text as embeddings and perform semantic similarity search, making them a key component of modern AI applications such as Retrieval-Augmented Generation (RAG), chatbots, and intelligent search systems.

---

## 🚀 Features

* Create and manage a ChromaDB collection
* Store documents as vector embeddings
* Perform semantic similarity search
* Retrieve the most relevant documents
* Learn the fundamentals of vector databases
* Beginner-friendly Jupyter Notebook implementation

---

## 🛠️ Tech Stack

* ChromaDB

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/codinghub37/vector-db-basics-chromadb.git
```

### 2. Navigate to the Project Directory

```bash
cd vector-db-basics-chromadb
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

Launch Jupyter Notebook:

```bash
jupyter notebook
```

Open **vector-db-basics.ipynb** and run all cells.
